## Description

This project is a **Payroll System for Rinku Cinemas**, developed as part of a **technical test**.  
It consists of:  

- **Backend API:** built with **Node.js, Express, and Sequelize**, connecting to a **MySQL** database  
- **Frontend Web:** built with **React, Material UI, and TailwindCSS**

## Installation

1. Clone the repository:

```bash
git clone <repository_url>
``` 

2. Install backend dependencies:

```bash
cd backend
npm install
``` 

3. Configure backend environment variables in .env:
```bash
# Server configuration
PORT=
NODE_ENV=

# Database configuration
DB_HOST=
DB_PORT=
DB_NAME=
DB_USER=
DB_PASSWORD=
``` 

4. Start the backend server:

```bash
npm run dev
``` 

5. Install frontend dependencies and start the frontend:
```bash
cd frontend
npm install
npm start
```
